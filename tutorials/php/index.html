The Basics

In this tutorial we will learn the different character types and how they are used. This tutorial will follow standardization known as PHP Standards Recommendations (PSR), found here.

Basic file structure

In PSR-1 we can use either <?php or <?= to open a PHP document. However, for this tutorial we will stick to using <?php

  
                            
                                <?php

                                

                                ?>

Above is an opened and closed PHP document, of course the file extension must be .php.

PHP supports the following character types: String, Integer, Boolean, Float, Array, Object, and NULL.

Variables

A variable is a value that can change. Its opposite is a constant.

Strings


A string is a series of characters, a string is determined by wrapping characters in either double-quotation or single-quotation marks.

    
                            
                                <?php

                                $String = "I'm a string!";

                                $StringToo = 'I am also a string'.;

                                ?>

Booleans and Numbers


Booleans are a logic device, the value is either TRUE or FALSE.

    
                            
                                <?php

                                $booleanTrue = TRUE;

                                $booleanFalse = FALSE;

                                ?>

PHP, being based on C languages, has a few options for holding numbers. The first of these, an integer, is any whole number. The second is a float. PHP does not differentiate between floats and doubles. The third is array, then objects, nulls, and resources.

    
                            
                                <?php

                                $integer = 1;

                                $float = 1.1;

                                $array = array();

                                $object = new foo();

                                $objectCall = $object->bar();

                                $null = NULL;

                                $resource = file_get_contents();

                                ?>
