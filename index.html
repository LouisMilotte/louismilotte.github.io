jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

www.jQuery.com

jQuery is the paint brush of scripting. The animations you saw and the popup this text is in is all powered by jQuery. With it we can communicate with servers without forcing users to new pages. We can change content simply on how the user interacts with your website. It is also what gives Twitter Bootstrap the amazing mobile friendly abilities it has.
